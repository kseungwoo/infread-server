## 커밋 컨벤션
- feat: 새로운 기능 추가
- fix: 버그 수정
- docs: 문서 수정
- style: 코드 포맷 변경, 코드 수정이 없는 경우
- refactor: 코드 리팩토링
- test: 테스트 코드 작성, 테스트 코드 리팩토링(프로덕션 코드 변경X)
- dev: 라이브러리 설치 등 개발 환경 세팅
- comment: 필요한 주석 추가 및 변경
- remove: 파일을 삭제하는 작업만 수행한 경우
- rename: 파일 혹은 폴더명을 수정하거나 옮기는 작업만 한 경우
- chore: 이외 기타 사항
